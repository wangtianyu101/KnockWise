> 合并自旧文档 `平台重构方案.md` + `前端重构方案.md`（2026-06 整合）。

## 一、演进背景

### 起点（V0）
KnockWise = AI 模拟面试工具，单模块。

### 目标（V1）
扩展为三模块个人 AI 工作台：
- 🎯 面试练习（已有基础）
- 📚 知识管理（新增：Obsidian 集成）
- 📡 信息推送（新增：AI 日报 / 周报 / 代码统计）

### 一句话定位
> 程序员的 AI 副脑 —— 帮你准备面试、管理知识、追踪信息。

### 命名
项目目录名保持 `Intervue`（避免破坏 git 历史），UI 品牌名定为 **`KnockWise`**（开发者的 AI 副脑）。

## 八、关键 API 增量

Phase 1-4 累计新增的接口（已全部落地）：

| 模块 | 接口 | 用途 |
|---|---|---|
| 认证 | `POST /api/auth/register` `POST /api/auth/login` | 邮箱 + 密码 |
| 认证 | `GET /api/auth/dev-login` | 本地跳过认证 |
| 面试 | `GET /api/interviews?status=&page=&size=` | 列表 + 分页 + 筛选 |
| 面试 | `POST /api/interviews/{id}/complete` | 结束面试 + 算分 |
| 报告 | `POST /api/reports/interview/{id}` | LLM 生成 11 维雷达 |
| 分析 | `GET /api/analytics/overview` | 综合统计 |
| 分析 | `GET /api/analytics/radar` | 11 维雷达数据 |
| 分析 | `GET /api/analytics/trends` | 趋势对比 |
| 分析 | `GET /api/analytics/recommendations` | AI 推荐练习 |
| 知识 | `GET /api/knowledge/{tree,search,note,graph,stats,backlinks}` | Obsidian 集成 |
| 信息 | `GET /api/news/{daily,weekly,stats,sources}` | AI 日报 / 周报 |
| 简历 | `POST /api/profile/resume` | PDF → LLM 提取技能栈 |

完整说明 + Request/Response 样例见 [`接口文档.md`(../20-参考/接口文档.md)。
