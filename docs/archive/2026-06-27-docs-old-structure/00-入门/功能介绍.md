# KnockWise — 功能介绍

> 个人 AI 工作台。面试练习 + 知识管理 + AI 信息推送，三模块一站。

---

## 产品定位

**为程序员打造的 AI 副脑**。不是另一个 ChatBot，而是三个垂直场景的深度整合：

1. **面试练习** — AI 追问引擎，模拟真实面试官
2. **知识管理** — Obsidian vault 集成，全文检索 + 知识图谱
3. **信息推送** — AI 日报/周报 + 代码统计，自动投递

---

## 模块一：面试练习

### 核心功能

| 功能 | 说明 |
|---|---|
| 🤖 AI 追问引擎 | 不是静态题库——根据你的回答动态追问，模拟真实面试压力 |

## 技术特色

- **追问引擎**：LangGraph 多 Agent 协作（选题 → 追问 → 评估 → 报告），不是模板驱动
- **全链路语音**：LiveKit WebRTC → WhisperLive STT → Agent → Piper TTS，< 3 秒延迟
- **零外部依赖**：Obsidian 直读、本地 Whisper、本地 TTS，数据不出本机
- **玻璃态 UI**：深色 OLED 主题 + 磨砂玻璃卡片 + Fira Code 等宽字体

## 登录方式

- 邮箱 + 密码注册/登录（PBKDF2-SHA256）
- GitHub OAuth
- Dev Login（本地开发跳过认证）
