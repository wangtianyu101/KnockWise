#### 状态机

```
加载题目 → 写答案（每 5s 存草稿到 localStorage）
                    │
                    ▼ 提交
                 AI 评分中（skeleton + 3-5s 等待）
                    │
                    ├─ 答对 + 自动 mastered → 绿色 toast
                    ├─ 答对 + learning → 蓝色 toast
                    ├─ 答错 → 红色 toast + 自动加错题
                    └─ 网络失败 → "提交失败 [重试]"
                            │
                            ▼
                    状态自动更新（侧栏 + 状态按钮同步）
                            │
                            ▼ "下一题"
                    拉下一道（SRS 队列 → 今日推荐 → 未练）
```

#### API 端点

| 时机 | 端点 |
|---|---|
| 页面挂载 | `GET /api/learn/questions/{qid}` |
| 提交 | `POST /api/learn/questions/{qid}/answer` |
| 改状态 | `PATCH /api/learn/progress/{qid}` |
| 收藏 | `PATCH /api/learn/progress/{qid}` `{is_bookmarked: true}` |
| 提交历史 | `GET /api/learn/progress`（按 qid 过滤） |
| AI 讲解 | `POST /api/qa/explain`（调用问答 service） |

#### 上下文注入规则

> **关键决策**：问答**不查知识库**。如果用户想问"我笔记里写过什么"，他应该去知识库搜，不是在学习复习里。

## 九、状态机总览

### 9.1 答题流（学 + 复习共用 `/learn/[qid]`）

```
[加载题目] → [写答案] → [提交] → [AI 评分中] → [显示结果]
                                              │
                                              ├─→ 答对 + mastered → 绿色 toast
                                              ├─→ 答对 + learning → 蓝色 toast
                                              ├─→ 答错 → 红色 toast + 加错题
                                              └─→ 网络失败 → "提交失败 [重试]"
```

### 9.2 复习流（`/review` → `/learn/[qid]` 模式）

```
[进 /review] → [看到期 5 道] → [开始复习] → [/learn/[qid]?mode=review&idx=1/5]
                                                  │
                                                  ▼ 答完
                                              [自动加载下一道]
                                                  │
                                              [5 道答完]
                                                  │
                                                  ▼
                                              [回 /review, 更新统计]
```

### 9.3 问答流

```
[进 /qa] → [选历史会话 或 + 新会话] → [输入问题] → [调 /api/qa/chat]
                                                              │
                                                              ▼
                                              [AI 响应（含引用）]
                                                              │
                                                              ▼
                                              [多轮对话]
                                                              │
                                              ▼ 退出
                                              [会话保存到 DB]
```

## 十、错误状态展示（统一）

| 场景 | 展示 |
|---|---|
| API 401 | 全屏"登录已失效" 模态 |
| API 500 | 顶部红条 + [刷新] |
| LLM 超时 | toast "AI 响应超时，请重试" |
| LLM 解析失败 | toast "AI 解析失败，请参考人工答案" |
| 网络断开 | toast "网络断开，答案已存草稿" |
| 空数据 | "还没有 X，开始第一个吧 [开始 →]" |
